# `backend/data/`

Seed inputs and bundled content for the backend. See
[docs/06-wikipedia-ingestion.md](../../docs/06-wikipedia-ingestion.md).

| File | Purpose | Status |
|------|---------|--------|
| `un_leaders.json` | Committed Wikidata snapshot of UN-country leaders (HoS + HoG QIDs), the default seed input. Shape: `[{ "qid": "Q567", "country": "Germany" }]`. | **Placeholder `[]`** — generated by the M2 SPARQL step ([06](../../docs/06-wikipedia-ingestion.md) §Step 1). |
| `seed_extra.json` | Optional hand-picked humans to seed beyond UN leaders. Same shape as `un_leaders.json`. | Empty `[]`. |
| `humanity_prompts.json` | Rotating oath + control prompts for the dissent-based humanity check (R12). | Authored; expand freely. |

To (re)generate `un_leaders.json`, run the SPARQL query in
[docs/06-wikipedia-ingestion.md](../../docs/06-wikipedia-ingestion.md) §Step 1
against `https://query.wikidata.org/sparql` and write the deduped person QIDs
here. Re-seeding with `EAGORA_SEED=force` upserts on `wikidata_id` and never
resets ratings or vote history.
